TorusNet
Reasoning engine

Places 15,000 agents — twenty specialist cortices of 750 tiles each — on the surface of a 3D torus. Opposite edges wrap, so every node has four neighbors and no boundary. Helical field lines (Pₓ=2, Pᵉ=1) thread sparse wavelet packets across the mesh.

FusionCoreNet
Control engine

A control-theory engine. Each cycle it reads pressure (β = load ÷ capacity) and disagreement (ε = dissent + congestion + overload). When instability rises it issues repair commands — throttle, reroute, verify, ratify — keeping all 15,000 agents coherent. It never answers the user.
